
# 📖 通用教学平台使用指南 (User Guide)

欢迎使用通用教学平台！本指南将帮助教师和学生快速掌握平台的核心功能。

## ✅ 功能开放策略（模块化）

平台功能分为两类：

- **通用能力（所有用户可用）**：AI 答疑、代码编辑（若启用）、学情分析、课程管理、作业/测验/资源等。
- **课程专属模块（按选课开放）**：如物理仿真课程的仿真与数值计算模块、写作课程的写作分析与润色模块等。

> 例：学生选修了“物理仿真课程”后，会解锁该课程的仿真与公式类工具；AI 答疑、代码编辑、学情分析等则对所有课程用户开放。

## 👨‍🏫 教师指南 (For Teachers)

### 1. 课程管理
- **创建课程**: 登录后点击 "创建课程"，填写课程名称、代码和学期信息。
- **章节管理**: 在课程详情页，可以添加、编辑和发布课程章节。
- **发布公告**: 使用公告功能向全班学生发送重要通知。

### 2. 写作教学（仅写作类课程开放）
- **布置作业**：发布写作任务（如文献综述、课程论文）。
- **查看提交**：在“写作提交”页面查看学生提交。
- **教学看板**（可选）：查看班级写作薄弱点与重点辅导对象。

### 3. 学生管理与考勤
- **考勤管理**: 发起课堂签到，实时查看出勤率。
- **学情监控**: 查看学生的学习进度和活跃度统计。

---

## 👨‍🎓 学生指南 (For Students)

### 1. 课程学习
- **加入课程**：查看已选修的课程列表。
- **章节学习**：浏览课程内容，完成章节打卡。
- **课程工具**：根据所选课程开放不同工具（例如仿真、实验、写作分析等）。

### 2. 学术写作助手（仅写作类课程开放）
- **提交作业**：在“写作提交”页面提交各类写作任务。
- **✨ 智能润色（Text Polishing）**：
    1. 进入“写作提交”→“润色助手”。
    2. 粘贴你的段落。
    3. 点击“开始润色”，AI 将提供：
        - 语法修正
        - 学术语气优化（Academic Tone）
        - 修改建议与对比说明
    4. 参考建议优化你的文章。

### 3. AI 答疑（通用能力）
- **智能助教**：在“AI 答疑”页面与助教对话，支持多学科知识问答与学习辅导。

## 🛠️ 常见问题 (FAQ)

**Q: 润色功能支持多长的文本？**
A: 建议每次润色 500 词以内的段落，以获得最佳效果。

**Q: 课程专属工具无法加载怎么办？**
A: 请检查网络连接，或确认是否已选修该课程；如仍有问题，请联系课程教师或助教。

**Q: 如何修改已提交的作业？**
A: 在作业截止日期前，您可以多次重新提交，系统以最后一次提交为准。
