# 创建第一门课程

> 本教程将指导您创建一门课程，以《电磁场与电磁波》为例。

## 前提条件

- 已完成[快速开始](../01-getting-started/quick-start.md)中的环境搭建
- 拥有教师角色账号

## 步骤

### 1. 登录系统

使用教师账号登录系统。

### 2. 创建课程

1. 点击左侧菜单 **课程管理**
2. 点击 **创建课程** 按钮
3. 填写课程信息：

| 字段 | 示例值 |
|------|--------|
| 课程名称 | 电磁场与电磁波 |
| 课程代码 | EMF2025 |
| 学期 | 2025-2026-1 |
| 课程描述 | 本课程介绍电磁场的基本理论，包括静电场、恒定电流场、静磁场、时变电磁场等内容。 |

4. 点击 **保存**

### 3. 配置课程模块

课程创建后，可以启用以下模块：

- **AI 助教**: 智能问答和学习辅导
- **作业管理**: 作业发布和批改
- **测验系统**: 在线测试
- **仿真实验**: 电磁场可视化仿真 (课程专属)

### 4. 添加学生

方式一：通过邀请码
1. 在课程详情页获取 **邀请码**
2. 将邀请码分享给学生
3. 学生使用邀请码加入课程

方式二：批量导入
1. 准备学生名单 CSV 文件
2. 点击 **批量导入** 上传文件

## 验证

- 学生列表中显示已加入的学生
- 课程模块页面可正常访问
