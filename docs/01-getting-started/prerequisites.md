# 环境要求

> 运行本项目所需的软件和环境配置

## 必需软件

| 软件 | 版本要求 | 说明 |
|------|----------|------|
| Docker | 24.0+ | 容器运行时 |
| Docker Compose | 2.20+ | 容器编排 |
| Node.js | 20.x LTS | 前端构建 |
| Go | 1.24+ | 后端编译 |
| Python | 3.9+ | AI/仿真服务 |

## 可选软件

| 软件 | 用途 |
|------|------|
| MySQL Client | 数据库调试 |
| Redis CLI | 缓存调试 |

## 硬件要求

### 开发环境
- CPU: 4 核+
- 内存: 8GB+
- 磁盘: 20GB+

### 生产环境
- CPU: 8 核+
- 内存: 16GB+
- 磁盘: 100GB+ (SSD 推荐)

## 网络要求

- 访问 Docker Hub / GitHub
- 访问 LLM API (如 OpenAI)
