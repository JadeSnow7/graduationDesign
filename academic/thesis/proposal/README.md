## 开题报告要点

### 项目概述
基于企业微信与大模型的高校《电磁场》课程智能教学平台设计与实现

### 主要创新点
1. 企业微信 + H5 的教学平台部署方式
2. 大模型与教学业务解耦设计
3. AI + 仿真协同教学模式
4. 多角色精细化权限体系

### 技术栈
- 前端：Vue 3 + Vite (H5)
- 后端：Go + Gin
- AI服务：Qwen 大模型
- 仿真：Python (NumPy/SciPy/Matplotlib)
- 数据库：MySQL
