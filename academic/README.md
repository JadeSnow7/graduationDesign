# 学术材料 (Academic Materials)

本目录包含与毕业设计相关的所有学术文档和研究材料。

## 目录结构

- **[thesis/](./thesis/)** - 毕业论文相关文件
  - 开题报告、论文草稿、最终版本、答辩材料
- **[reports/](./reports/)** - 各类报告
  - 双周报告、进度报告、实验报告
- **[literature/](./literature/)** - 相关文献
  - 原始论文PDF、翻译文档、阅读笔记
- **[presentations/](./presentations/)** - 演示材料
  - 答辩PPT、项目展示、演讲稿

## 论文写作

- LaTeX源码位于 `thesis/src/`
- 编译产物位于 `thesis/build/`
- 使用 `latexmk` 进行编译

## 文献管理

- 原始论文存放在 `literature/papers/`
- 翻译文档存放在 `literature/translations/`
- 参考文献库: `literature/bibliography.bib`

## 进度跟踪

- 双周报告记录研究进展
- 实验数据和结果分析
- 技术难点和解决方案记录
