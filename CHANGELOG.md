## [0.1.0] - 2024-12-23

### 新增
- 初始项目结构

- 后端 Go 服务
- AI 服务 Python 应用
- 仿真服务 Python 应用

### 技术栈
- 前端：Vue.js 3 + TypeScript + Vite

- 仿真服务：Python + FastAPI
- 数据库：MySQL
- 容器化：Docker + Docker Compose
- 反向代理：Nginx

## 变更类型说明

- `新增` - 新功能
- `变更` - 现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - 问题修复
- `安全` - 安全相关的修复

## 版本号规则

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

## 贡献指南

请在提交变更时更新此文档：

1. 在 `[未发布]` 部分添加你的变更
2. 使用适当的变更类型分类
3. 提供清晰简洁的变更描述
4. 在发布时将变更移动到新的版本部分

## 发布流程

1. 更新版本号
2. 将 `[未发布]` 的变更移动到新版本
3. 添加发布日期
4. 创建 Git 标签
5. 发布到相应平台
