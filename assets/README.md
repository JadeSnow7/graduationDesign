# 静态资源 (Assets)

本目录包含项目使用的各种静态资源文件。

## 目录结构

- **[images/](./images/)** - 图片资源
  - 项目截图、用户界面图片、图标等
- **[diagrams/](./diagrams/)** - 架构图和流程图
  - 系统架构图、数据流图、UML图等
- **[templates/](./templates/)** - 模板文件
  - 文档模板、配置模板、代码模板

## 文件规范

### 图片文件
- 格式: PNG (截图), SVG (图标), JPG (照片)
- 命名: 使用描述性英文名称，小写+连字符
- 大小: 控制在合理范围内，避免过大文件

### 架构图
- 使用 Mermaid 格式 (.mmd) 便于版本控制
- 同时提供 PNG/SVG 导出版本
- 包含图例和说明文字

### 模板文件
- 提供常用的文档和配置模板
- 包含详细的使用说明
- 保持模板的通用性和可复用性

## 使用指南

1. 在文档中引用图片时使用相对路径
2. 大型媒体文件考虑使用外部存储
3. 定期清理不再使用的资源文件
4. 为重要图片提供替代文本描述
